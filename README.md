# ecs
A Go-implementation of the ECS (Entity-Component-System).

ECS engine for RockGO

Need go1.18 for generics code

## Install

how to install go1.18 ?
```
$: go get golang.org/dl/gotip
$: gotip download
```
how to get ecs?
```
$: go get -u github.com/zllangct/ecs
```
